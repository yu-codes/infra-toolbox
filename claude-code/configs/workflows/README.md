# Workflow Configs

Claude Code Dynamic Workflows 的配置與範例。

## 檔案說明

| 檔案 | 說明 |
| ---- | ---- |
| `settings-workflow.json` | 啟用/停用 workflow 的 settings 範本 |
| `example-workflow.md` | 自訂 workflow 的使用範例 |

## 快速使用

將 `settings-workflow.json` 的內容合併到你的 `~/.claude/settings.json` 或專案的 `.claude/settings.json`。
